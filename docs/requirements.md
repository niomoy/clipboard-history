# 需求文档 — 历史粘贴板

## 项目简介

一款 Windows 剪贴板历史管理工具，后台自动记录用户复制的文字和图片，支持随时回溯查找并再次粘贴。

## 目标用户

普通 Windows 用户（非技术人员），期望简洁直观的操作体验。

## 软件形态

| 形态 | 说明 |
|------|------|
| 系统托盘图标 | 常驻 Windows 右下角通知区域，点击图标弹出历史面板 |
| 独立完整窗口 | 可独立操作的完整窗口，支持搜索、置顶、删除等功能 |
| 开机自启 | 随 Windows 启动自动在后台静默运行 |
| 后台静默 | 平时只在托盘区显示小图标，不打断用户工作 |

## 功能需求

### F1 — 自动记录剪贴板
- 后台监听 Windows 剪贴板变化
- 支持文字内容（文本）
- 支持图片内容（截图、复制图片等）
- 去重：连续复制相同内容不重复记录

### F2 — 卡片式展示
- 每条记录一张卡片
- 按时间降序排列（最新在上）
- 置顶记录始终排在最前
- 图片记录显示缩略图

### F3 — 缩略图预览
- 图片记录自动生成缩略图
- 缩略图显示在卡片上，方便用户识别

### F4 — 点击即复制
- 点击卡片 → 内容自动回到系统剪贴板
- 用户随后在任意位置 Ctrl+V 粘贴
- 文字和图片均支持此操作

### F5 — 置顶
- 重要记录可置顶
- 置顶记录固定在列表顶部
- 置顶记录不受过期自动清理影响

### F6 — 删除
- 每条记录可手动删除
- 删除文字记录：从数据库移除
- 删除图片记录：同时删除图片文件和缩略图文件

### F7 — 搜索
- 顶部搜索栏，输入关键词实时过滤
- 仅搜索文字内容（图片记录不会被搜索过滤掉，或按类型筛选）

### F8 — 保留天数
- 用户可设置保留天数：1天 / 3天 / 5天
- 超过保留天数的普通记录自动清理
- 置顶记录永不过期
- 程序启动时及运行中定期执行清理

## UI/UX 要求

- 简洁直观，无需学习成本
- 主色调：淡蓝色系
- 卡片式布局，信息层次清晰
- 窗口大小适中，支持滚动浏览

## 非功能需求

- 最终交付为单个 .exe 文件，无需安装依赖
- 支持 Windows 10/11
- 内存占用小，后台静默运行不影响性能
