# UI 设计规范 — 历史粘贴板

## 设计原则

- **简洁**：减少视觉噪音，用户一眼看到核心功能
- **直观**：无需说明书即可使用
- **轻量**：窗口不臃肿，颜色柔和

## 配色方案

| 用途 | 色值 | 预览 |
|------|------|------|
| 主背景 | `#FFFFFF` | 纯白 |
| 卡片背景 | `#E3F2FD` | 淡蓝 |
| 卡片悬停 | `#BBDEFB` | 稍深蓝 |
| 置顶卡片背景 | `#BBDEFB` | 稍深蓝（与悬停同色） |

| 主文字色 | `#333333` | 深灰 |
| 次要文字色 | `#888888` | 浅灰（时间戳） |

## 卡片组件规范

### 文字卡片
- 高度：自适应（最小 60px）
- 内边距：12px
- 文字内容最多显示 3 行，溢出显示省略号
- 底部显示时间戳（左侧）+ 操作按钮（右侧）
- 操作按钮：置顶（📌）、删除（🗑）

### 图片卡片
- 高度：自适应（约 150px）
- 左侧：缩略图（160×120px，保持比例填充）
- 右侧：占位文字 "图片" 或留白
- 底部显示时间戳 + 操作按钮
- 点击图片区域可放大预览

### 卡片交互
- 鼠标悬停：背景色变为 `#BBDEFB`
- 鼠标点击卡片主体：复制内容到剪贴板 → 短暂显示 "已复制✓" 提示
- 置顶按钮点击：切换置顶状态 → 卡片移至顶部/归位
- 删除按钮点击：弹出确认 → 删除记录

## 搜索栏规范

- 占位文字："搜索历史内容..."
- 输入时实时过滤（无需按回车）
- 无匹配结果时显示 "无匹配结果"
- 搜索框右侧有清空按钮（×）

## 设置窗口规范

- 模态窗口，宽度 320px
- 标题："设置"
- 内容：
  - 保留天数：下拉选择（1天 / 3天 / 5天）
  - 开机自启：勾选框
- 底部：保存按钮 + 取消按钮

## 字体

- 默认字体：Microsoft YaHei（微软雅黑）
- 卡片标题/内容：13px
- 时间戳：11px
- 搜索框：13px
